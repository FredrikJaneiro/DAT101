  1. [In this assignment, we use the DOM. Explain how this library can help us develop interactive web pages.]

    2. [The DOM (Document Object Model) allows us to manipulate HTML and CSS elements dynamically, enabling interactive web pages by updating content, styling, and structure in response to user actions or events.]

    <2i>[What is an event? Simply explain what happens when we add an event listener to an object.]

    [An event is an action that occurs in the browser, such as a click or keypress. When we add an event listener to an object, we specify a function that will be executed when that event occurs.]

    <3i>[Sometimes we need to use an event object to get data from the event. Explain how this event object is structured, don't explain all functions or attributes. Try to explain, for example, how to retrieve data from a keyboard or mouse.]

    [The event object contains properties and methods related to the event that occurred. For example, when a key is pressed, the event object has a `key` property that holds the character of the key pressed. Similarly, for mouse events, it has properties like `clientX` and `clientY` to get the position of the mouse click.]

    <4i>[Explain the main difference between DOM id and DOM name. Can we have identical IDs?]

    [The main difference is that an ID must be unique within a document, while a name can be shared among multiple elements. No, we cannot have identical IDs in a single HTML document.]
