1. Where should you use constants, and where should you use variables? Can they all be constants?

    2. [You should use constants when the value does not change throughout the program, and variables when the value can change. Not all values can be constants.]

  2. How do parentheses affect the arithmetic outcome of an expression?

    3. [Parentheses change the order of operations, ensuring that the expressions within them are evaluated first.]

    4. [We can use the toFixed() method to limit floating-point numbers to a specific number of decimal places.]

    5. [We can compare two floating-point values by checking if the absolute difference between them is less than a small threshold value (epsilon).]

  5. What do comparison operators return, and how can we check if they are FALSE or TRUE?

    6. [Comparison operators return boolean values (TRUE or FALSE). We can check their values directly in conditional statements or by comparing them to TRUE or FALSE.]

  6. How does the Math.random() function work, and what can it be used for?

    7. [The Math.random() function generates a pseudo-random floating-point number between 0 (inclusive) and 1 (exclusive). It can be used for various purposes, such as simulating randomness in games, selecting random items from a list, or generating random test data.]

  7. What is modulus (%), and when do we need this operator in mathematical operations?

    8. [The modulus operator (%) returns the remainder of a division operation. It is useful for determining if a number is even or odd, cycling through a set of values, or constraining values within a certain range.]
